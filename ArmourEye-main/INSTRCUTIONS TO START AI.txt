INSTRCUTIONS

1. Start the local infrastructure

Docker Desktop

Launch Docker Desktop and wait until it says “Docker is running.”
If you plan to use docker-compose, run from repo root:

cd <your-path-to>/ArmourEye-main
docker compose up -d

If you don’t use compose, just make sure Docker is on; the backend spins up containers on demand.



Backend API (Node)

New PowerShell window:

cd <your-path-to>/ArmourEye-main/backend
npm install   # first time only
npm run start

Leave this running; it serves http://localhost:3001.



Frontend (Vite React app)
Another PowerShell window:

cd <your-path-to>/ArmourEye-main
npm install   # first time only
npm run dev

Visit the URL shown (usually http://localhost:5173).

Install Python 3.11 (from python.org) and ensure it’s on PATH.
Install Microsoft Visual C++ Build Tools (Desktop development with C++ workload) so pip can compile packages like hnswlib.

Create a virtual environment in the repo root:

cd <your-path-to>/ArmourEye-main
python -m venv .venv-ai
.venv-ai\Scripts\activate   
python -m pip install --upgrade pip

Install dependencies inside that venv (GPU build of PyTorch plus the AI stack):
   
pip install torch --index-url https://download.pytorch.org/whl/cu121   
pip install fastapi uvicorn[standard] transformers accelerate bitsandbytes langchain-community sentence-transformers chromadb sentencepiece

Launch the inference server:
   
uvicorn backend.ai.inference.server:app --host 0.0.0.0 --port 8000

Leave this terminal open while using the app.
